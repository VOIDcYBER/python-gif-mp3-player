# پلیر GIF و موسیقی

یک اسکریپت ساده هست که گیف و اهنگ رو از سایت دانلود میکنه  و اون رو پخش میکنه

## ویژگی‌ها
- دانلود گیف و اهنگ از سایت
- نمایش گیف با برنامه پیشفرض سیستم
- pygame پخش اهنگ با کتابخانه 

## پیش‌نیازها
- Python 3.x
- کتابخانه `pygame`
- کتابخانه `requests`

## نصب
```bash
pip install pygame requests
